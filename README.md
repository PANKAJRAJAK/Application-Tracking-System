# ATS Resume Scanner

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/your-username/ats-resume-scanner)

A powerful and user-friendly web application built with Streamlit for analyzing resumes against job descriptions using advanced AI. This Applicant Tracking System (ATS) Resume Scanner helps job seekers and recruiters evaluate resume-job fit efficiently.

## 🚀 Features

- **Resume Upload**: Easily upload PDF resumes for instant analysis.
- **Job Description Input**: Paste or type job descriptions directly into the app.
- **Comprehensive Analysis**:
  - **Profile Evaluation**: Get detailed feedback on strengths and areas for improvement.
  - **Keyword Extraction**: Identify key skills and terms to optimize your resume.
  - **Match Percentage**: Receive a percentage score of resume-job alignment, including missing keywords and actionable insights.
- **AI-Powered Insights**: Provides intelligent analysis for resumes and job descriptions.
- **User-Friendly Interface**: Intuitive design for quick and easy use.

## 📋 Prerequisites

- Python 3.8 or higher

## 🛠 Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/PANKAJRAJAK/Application-Tracking-System.git
   cd Application-Tracking-System
   ```

2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Set Up Environment Variables**:
   - Create a `.env` file or set environment variables if required by your environment.

4. **Run the Application**:
   ```bash
   streamlit run app.py
   ```

   The app will open in your default web browser at `http://localhost:8501`.

## 📖 Usage

1. **Launch the App**: Follow the installation steps to start the Streamlit server.
2. **Enter Job Description**: Use the text area to input the target job description.
3. **Upload Resume**: Click the upload button to select and upload your PDF resume.
4. **Analyze**:
   - Click "Tell Me About the Resume" for a detailed profile evaluation.
   - Click "Get Keywords" to extract essential skills.
   - Click "Percentage Match" for a compatibility score and recommendations.
5. **Review Results**: Analyze the AI-generated insights to improve your resume or assess candidates.
